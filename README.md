# AI-Driven Hybrid Continuous Trust Scoring Engine
## Overview

This project explores the impact of AI on the cyber offense–defense balance by implementing a hybrid continuous trust scoring system for adaptive access control.

The model combines:

* Isolation Forest anomaly detection
* Rule-based risk augmentation
* Weighted hybrid risk fusion
* Threshold-based adaptive decision logic

## Key Features

* Continuous risk scoring (0–100 scale)
* Hybrid ML + deterministic rule integration
* Threshold calibration analysis
* Performance comparison (ML-only vs Hybrid)
* Visualization of risk distributions and confusion matrix

## Dataset

UNSW-NB15 intrusion detection dataset

## Architecture

Traffic → Feature Processing → ML + Rule Engine → Hybrid Risk Fusion → Adaptive Access Control → Oversight
